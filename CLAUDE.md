# NBA球星卡市场走势软件

一款面向NBA球星卡收藏者的市场价格追踪与持仓比价网页应用。

- 会话结束：记录完成事项和新的待办事项

## 工作指引

### 核心原则
1. **小步推进** — 每次只做一个阶段中的一个小任务，确认稳定后再继续
2. **文档先行** — 需求变更必须先更新 `docs/` 下的对应文档
3. **日志跟踪** — 每次会话更新 `dev-logs/`，不过夜

### 技术约束
- 前端：React + TypeScript + Vite + TailwindCSS
- 后端：Node.js + Express + sql.js（SQLite）
- 配色：参考 `docs/design-spec.md` 中的色值，不要自行发挥
- 数据：eBay和卡淘的抓取模块需注意反爬策略，频率控制

### 启动步骤（每次开发会话）
1. 查看 `dev-logs/` 最新日志，了解当前进度
2. 查看 `docs/execution-plan.md` 确认当前阶段
3. 开始工作
4. 结束时更新日志

### 用户特点
- 非技术背景，交互必须简洁直观
- 收藏规模小（<50张），功能不需要过度设计
- 预算敏感，优先低成本方案
