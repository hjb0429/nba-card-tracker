# NBA球星卡市场走势软件 — 设计规范

## 品牌定位
湖人紫金主题，低饱和度、清爽、专业中带活力。

## 配色方案

### 主色调
| 用途 | 色值 | 说明 |
|------|------|------|
| 主紫 | `#8B7EC8` | 按钮、标题、重点区域 |
| 主金 | `#E8D5A3` | 点缀、高亮、徽章 |
| 深紫 | `#5B4A8C` | 文字、图标 |
| 浅金 | `#F5EDD6` | 卡片背景高亮 |

### 中性色
| 用途 | 色值 | 说明 |
|------|------|------|
| 背景 | `#FAF8FC` | 页面底色（淡紫调） |
| 卡片白 | `#FFFFFF` | 卡片/面板背景 |
| 文字主 | `#2D2D3F` | 正文文字 |
| 文字辅 | `#8E8EA0` | 辅助信息 |
| 边框 | `#E8E6F0` | 分隔线、边框 |
| 成功绿 | `#6BBF8A` | 价格上涨 |
| 警告红 | `#E88B8B` | 价格下跌 |

### 图表色
- 国内价格线：`#8B7EC8`（主紫）
- 海外价格线：`#E8D5A3`（主金）加深为 `#C9A84C` 保证可读性

## 字体
- 系统字体优先：`-apple-system, "Microsoft YaHei", "PingFang SC", sans-serif`
- 中文字体以微软雅黑、苹方为主
- 数字使用等宽变体：`"SF Mono", "Consolas", monospace`

## 布局

### 响应式断点
| 断点 | 宽度 | 适配设备 |
|------|------|---------|
| `sm` | < 640px | 手机竖屏 |
| `md` | 640px - 1024px | 手机横屏/平板 |
| `lg` | > 1024px | 桌面 |

### 页面结构
- 顶部固定导航栏（Logo + 三个Tab）
- 内容区最大宽度 1200px，居中
- 移动端底部Tab切换
- 卡片间距 16px

## 组件规范

### 卡片（Card）
- 白色背景，圆角 12px
- 阴影：`0 1px 3px rgba(91,74,140,0.08)`
- 内边距 20px

### 价格标签
- 金额使用等宽数字
- 涨跌用绿色/红色+箭头图标
- 字号加大，醒目

### 图表
- 折线粗细 2px
- 数据点 4px 圆
- 网格线淡色
- 无图例边框
