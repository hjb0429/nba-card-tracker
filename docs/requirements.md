# NBA球星卡市场走势软件 — 需求文档

## 项目概述
面向NBA球星卡收藏者，提供市场价格走势查询、个人持仓管理、比价分析功能的网页应用。

## 目标用户
- 小规模NBA球星卡收藏者（50张以内）
- 同时关注国内（卡淘）和国际（eBay）市场
- 非技术用户，需要简洁直观的界面

## 功能需求

### F1. 市场行情
- 按系列、球员、卡片型号筛选
- 四个时间维度：近1周 / 近1月 / 近3月 / 近1年
- 价格走势折线图
- 数据源：eBay已售成交价 + 卡淘成交价
- 首期覆盖主流系列：Prizm、Optic、Select、Hoops、Donruss、Contenders、National Treasures、Flawless、Mosaic、Revolution（共10个）

### F2. 我的持仓
- 手动录入球星卡信息（系列、年份、球员、型号、编号等）
- 支持上传卡片实物照片
- 列表展示，支持编辑和删除
- 每张卡自动关联最新市场参考价

### F3. 比价分析
- 持仓卡与当前市场价并排对比
- 显示涨跌幅度和趋势方向
- 标记可能的出手时机（价格高位区间）

### F4. 市场切换
- 国内/国外市场价格切换查看
- 双市场对比视图

## 非功能需求

### 性能
- 首屏加载 < 3秒
- 图表渲染 < 1秒

### 兼容性
- Windows Chrome/Edge
- iOS Safari
- Android Chrome

### 安全
- 用户数据隔离
- 图片上传大小限制（单张 ≤ 5MB）

### 成本
- 月度运营成本 ≤ 50元
- 优先免费/开源方案

## 不纳入首期
- App原生开发（后续扩展）
- 非主流系列卡片
- 自动交易/下单功能
- 社区/社交功能
